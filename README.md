## About this repository

This repository hosts my personal website as a single, self-contained `index.html` file — all HTML, CSS and the vector illustration live in that one file, with no build step or external dependencies (fonts are loaded from Google Fonts). It is served free via **GitHub Pages**.

### Tech notes

- Static single-page site: sticky navigation, responsive layout, dark theme.
- Accessible: keyboard focus states, reduced-motion support, WCAG AA colour contrast.
- SEO-ready: meta description, Open Graph tags and `schema.org` Person structured data.

### Running or editing locally

1. Download `index.html`.
2. Double-click it to open in any web browser — that's the full site.
3. Edit the file in any text editor, save, and refresh the browser to preview changes.

### Deployment

The site is published from the `main` branch (root) via **Settings → Pages**. Any commit to `index.html` updates the live site at https://mfalias.github.io within about a minute.
